# RA On Duty Schedule - Annexes

A web application displaying the Residence Assistant duty schedule for Trent University's Lady Eaton, Champlain, and Gzowski Annexes.

## Features

- **Two Schedule Options**: Toggle between 9-RA team (2 per night) and 3-Person team (1 per night)
- **Interactive Calendar View**: See daily assignments at a glance
- **Weekly Hour Breakdowns**: Track hours per RA each week
- **Monthly Commitment Summary**: View total shifts and hours per RA
- **Responsive Design**: Works on desktop and mobile

## Schedule Details

### 9-RA Team Schedule
- 2 RAs on duty each night
- 9-day rotation cycle
- 3-4 days off between shifts
- ~14 hours/month per RA

### 3-Person Team Schedule
- 1 RA on duty each night
- 3-day rotation cycle
- 2 days off between shifts
- ~21 hours/month per RA

## Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Deployment to Vercel

1. Push this code to your GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Sign in with GitHub
4. Click "New Project"
5. Import your repository
6. Click "Deploy"

That's it! Vercel will automatically detect it's a Next.js app and deploy it.
